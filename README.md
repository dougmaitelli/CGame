## Keyboard shortcuts

### Movement

* <kbd>A</kbd> : Back
* <kbd>S</kbd> : Drop
* <kbd>D</kbd> : Forward
* <kbd>Space</kbd> : Jump

### Actions

* <kbd>Left Shift</kbd> : Fire
